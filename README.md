# 프로젝트 명 : 헝크리 샤크 (Hungry Shark)🦈🦈🦈🦈🦈🦈🦈🦈
## RULE🦈🦈🦈🦈🦈🦈🦈🦈
- Java로 구현한 헝크리 샤크.
- 방향키(➡⬅⬆⬇↗↘↙↖) 움직이며 적들을 피해다니며 물고기들을 먹는 게임.
- 적(폭탄, 나보다 큰 상어), 물고기가 특정 시간마다 새로 생성.
- 물고기 or 나보다 작은 적을 먹으면 점수 획득 + 크기 증가. 
- 폭탄 or 나보다 큰 적과 부딪히면 HP감소 + 크기 감소. 
- HP가 0가 되면 게임종료.
- 왼쪽 아래 부스터 사용가능, 레벨당 1번씩.
